Chapter 5. Python

Example 5.1. The most basic way to install Python

sudo apt-get install python-setuptools
sudo easy_install pip
sudo pip pygments

Using the new Python notebooks requires a bit more software, there are many out there but we will use Jupyter, this is a popular software among Python enthusiast.

  1. Go to the Continuum website at https://www.continuum.io/downloads
  2. Download the Anaconda installer. At this point, you most likely have the Python 2.x version, so choose that for the download

  3. Follow the instruction on the website on how to install Anaconda, most likely, it simply be a bash command

    bash Anaconda2-4.2.0-Linux-x86_64.sh
  4. Install the Jupyter software using the command conda install jupyter

To start using the interactive notebook, create a working directory and start the notebook from there

Example 5.2. Starting a Python notebook

mkdir pythontest && cd pythontest
jupyter notebook

It will start a webserver and quite possibly, launch your default browser as well. Then you can start mucking around the notebook
