15.2. Procmail

Example 15.6. Procmail configuration

CORRECTHOME=/home/yourname/mail/in 
MAILDIR=$CORRECTHOME/ 
PMDIR=/home/yourname/mail/procmail/info/in 
LOGFILE=$PMDIR/log 
VERBOSE=on 

:0
Mail/

Path to where you would like the physical mail files to be stored. Substitute your actual username to yourname, if you’re not sure what that is, run whoami

Were just setting the the MAILDIR variable

Were just creating a Procmail home folder and saving it to the PMDIR variable

Set the log file

Set this to on when you are testing, you can set this to off later on when things are properly configured already. This could be useful when debugging so you can see which systems are failing

Pay attention to the slash at the end of the Mail, that is important
