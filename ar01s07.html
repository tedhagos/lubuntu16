Cloud Storage

  1. Get overgrive, setup with your Google account
  2. Download Dropbox, then sudo dropbox start -i
  3. Wait for a (long) while, when dropbox finishes downloading the installer, sync Dropbox
  4. Everytime you reboot, start dropbox manually sudo dropbox start. This is necessary because as of Yaketty, the dropbox daemon is broken, it doesn’t start automatically
