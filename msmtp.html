15.3. msmtp

Msmtp is an smtp client. It will be the one responsible for sending out email messages. In order to use it, we first have to get it via sudo apt-get install -y msmtp.

test it

Now

Example 15.7. Msmtp configuration

defaults
auth on
tls on
tls_trust_file /etc/ssl/certs/ca-certificates.crt

account Gmail
host smtp.gmail.com
port 587
from yourusername@gmail.com
user yourusername@gmail.com
password yourpassword

account default: Gmail
