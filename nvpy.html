Chapter 11. nvPY

nvPY is a note-taking application. It’s a python-based equivalent of Notational Velocity (OSX). It was used by a lot of people to sync with SimpleNote, but it can be configured to read from a Dropbox directory (or any directory for that matter).

sudo apt-get install nvpy

Create a config file for nvpy

touch ~/.nvpy.cfg

Create the nvpy configuration file on your home folder

nvpy.cfg.  

[nvpy]
notes_as_txt = 1
txt_path = /home/thagos/Dropbox/notes

#disable simplenote syncing, use 1 to enable it
simplenote_sync = 0

# 0 value will sort in alphanumeric mode
sort_mode = 1
